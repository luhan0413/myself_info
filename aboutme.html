    大學後期 (大三～大四)
    -人工智慧研發產學中心
    實習

    開發基於 YOLOv8 深度學習 的即時交通違規檢測平台，能自動辨識機車騎士未戴安全帽與超載行為，並整合車牌辨識 API 與違規紀錄管理系統。

    成果：YOLOv8 檢測準確率達 85%+，多執行緒架構將延遲降 70%，實現 30 FPS 即時影像處理，並以 Git 協作與敏捷開發流程完成系統整合。
